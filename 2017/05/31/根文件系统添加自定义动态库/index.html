根文件系统添加自定义动态库

发表于 2017-05-31 | 分类于 嵌入式

自定义的动态库最好不要放到根文件系统的必要目录lib下

创建制作库的.c和.h文件

例如

生成动态库

arm-cortex_a9-linux-gnueabi-gcc -shared -fpic -o libtest.so test.c

测试

执行：arm-cortex_a9-linux-gnueabi-gcc -o main main.c -L. -ltest
添加环境变量：export LD_LIBRARY_PATH=/home/applib:$LD_LIBRARY_PATH

/home/applib：库文件所在目录
