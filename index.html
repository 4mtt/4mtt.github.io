linux串口工具-kermit

发表于 2017-05-27

安装

sudo apt-get install ckermit

获取linux下的串口号

1：插入USB转串口线

2：执行dmesg,获取打印信息

3：根据串口设备文件名修改配置文件.kermrc

配置.kermrc

set line        /dev/ttyUSB0
set speed        115200
set carrier-watch     off
set handshake        none
set flow-control    none
robust
set file type        bin
set file name        lit
set rec pack        1000
set send pack        1000
set window        5

使用

sudo kermit -c

linux命令-tree

发表于 2017-05-27

tree :以树形形式列出当前目录下所有深度级别的文件和目录

安装

sudo apt-get install tree

示例

tree -L 2
查看当前第2级的目录和文件(tree -L N)

tree -L 2 > tree.txt
以树形形式保存到tree.txt中

deb格式软件安装

发表于 2017-03-27

deb是debian linux的安装格式,最基本的安装命令:
dpkg -i file.deb

dpkg是DebianPackage的简写,为Debian的套件管理系统.

用法

1:dpkg -i file.deb

安装一个Debian软件包

2:dpkg -c file.deb

列出软件包的内容

3:dpkg -I file.deb

提取包裹信息

4:dpkg -r file

移除已安装的软件包

5:dpkg -P file

完全清除已安装的软件包

6:dpkg -L file

列出file安装的所有文件清单

7:dpkg -s file

显示已安装软件包的信息

8:dpkg-reconfigure file

重新配置已安装的包裹,若他使用的是debconf(为包裹安装提供了一个统一的配置界面)
