BusyBox的诞生

  • BusyBox 最初是由 Bruce Perens 在 1996 年为 Debian GNU/Linux 安装盘编写的。
    其目标是在一张软盘上创建一个可引导的 GNU/Linux 系统，这可以用作安装盘和急救盘。
    一张软盘可保存约 1.4-1.7MB的内容，因此没有多少空间留给 Linux 内核及相关的用户应用程序使用。

制作img文件

发表于 2017-05-31 | 分类于 嵌入式

创建空白镜像文件

  • sudo dd if=/dev/zero of=rootfs_ext4.img bs=1k count=8196
    • dd:用于创建一个单个镜像文件
    • if=/dev/zero:将来创建的单个镜像文件里面的内容全部来自设备/dev/zero
    • of=rootfs_ext4.img:指定镜像名且此文件填充全0
    • bs=1k:生成的rootfs_ext4.img以块为单位,一块1024字节
    • count=8196:总共8196块

格式化

  • sudo mkfs.ext4 rootfs_ext4.img
  • 把rootfs_ext4.img比作U盘指定的文件系统格式为ext4

挂载

  • sudo mkdir /mnt/initrd
  • sudo mount -t ext4 -o loop rootfs_ext4.img /mnt/initrd
    • 挂接rootfs_ext4.img到目录/mnt/initrd,并指定文件系统类型ext4

写镜像

  • sudo cp /rootfs/* /mnt/initrd -frd
    • 向rootfs_ext4.img拷贝rootfs的内容
  • sudo umount /mnt/initrd
